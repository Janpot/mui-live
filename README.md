# mui-live

Visual CSS editor.

1. Install

   ```sh
   pnpm add mui-live
   ```

1. Add the plugin to your vite config

   ```tsx
   export default defineConfig({
     // ...
     plugins: [react(), live()],
     // ...
   });
   ```

1. Run your application

1. Open the mui-live browser

   ```sh
   pnpm dlx mui-live
   ```

1. Visit your application in the browser
