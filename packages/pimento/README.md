Visual CSS editor.

1. Install

   ```sh
   pnpm add pimento
   ```

1. Add the plugin to your vite config

   ```tsx
   export default defineConfig({
     // ...
     plugins: [react(), pimento()],
     // ...
   });
   ```

1. Run your application

1. Open the pimento browser

   ```sh
   pnpm dlx pimento
   ```

1. Visit your application in the browser
